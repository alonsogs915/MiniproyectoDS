Nuestro miniproyecto se basa en la creación de una página web diseñada para el análisis financiero de empresas, permitiendo obtener diversas métricas e información relevante para evaluar su desempeño. Hemos añadido secciones donde los usuarios pueden buscar empresas y acceder a noticias actualizadas sobre ellas. Para lograr esto, utilizamos APIs como Alpha Vantage y NewsAPI. Además, implementamos tecnologías como Next.js, Tailwind CSS, y JavaScript, entre otras, para ofrecer una experiencia fluida y visualmente atractiva.
